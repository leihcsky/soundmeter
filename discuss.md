## 📊 一、什么是分贝（dB）？

**分贝（dB）**是声音强度的单位，采用对数刻度，这意味着声音每增加 10 dB，声压级强度会约增加 10 倍，而主观感受上大约是“听起来两倍大声”。真实世界声音例子包括：

* **0 dB**：接近听觉阈值（几乎听不到）
* **30 dB**：轻声耳语
* **60 dB**：正常对话
* **80 dB**：繁忙街道、门铃
* **100 dB**：嘈杂工地、电吹风
* **110–120 dB**：接近耳边大叫、警笛
* **130+ dB**：飞机起飞、爆竹声（可能当场损伤） ([世界卫生组织][1])

---

## 📌 二、分贝与健康影响（权威指南）

### ✅ 1. **低噪音（< 70 dB）**

* 70 dB 及以下通常不会对听力造成显著剩余风险。
* 世界卫生组织（WHO）建议日常噪音尽量控制在 **70 dB 以下**，尤其是儿童环境（如教室 / 家庭环境），以减少长期影响风险。 ([Hearing Health Foundation][2])

📌 常见噪音：

* 图书馆、安静室内环境
* 轻声耳语

---

### ⚠️ 2. **中等噪音（70 – 85 dB）**

* 常见于繁忙车流、吸尘器、音乐播放器在较高音量下。
* 长时间持续在这个级别下（尤其每天长期暴露），也可能增加听力疲劳与听力病风险。
* NIOSH 建议 **85 dBA 作为日常职业噪音的推荐最大 8 小时暴露值**，超过这个水平风险显著增加。 ([疾病控制与预防中心][3])

📌 提示：

* 即使是在家，**持续高于 70 dB 的噪音累积**时间越长，对耳朵的压力越大。
* 例如在繁忙交通环境中讲话可能需要提高声量——这是身体对噪音掩盖的自然反应。 ([Hearing Health Foundation][2])

---

### 🛑 3. **高噪音（85 dB 及以上）**

这是典型的“听力伤害阈值”：

* **≥ 85 dBA**：长期每天 8 小时暴露可能导致噪音性听力损失（NIHL）。
* 更高级别下，允许的安全暴露时间迅速缩短：

  * ~90 dB：数小时内可损伤
  * 100 dB：十几分钟即可出现损伤迹象
  * 110 dB：几分钟内即有风险
  * 120 dB：几秒钟就可能造成即时伤害（如爆炸声）。 ([nidcd.nih.gov][4])

📌 实例：

| 声音强度     | 危害特征                  |               |
| -------- | --------------------- | ------------- |
| 85 dB    | 工厂 / 重交通噪音，8 h 暴露风险增高 |               |
| 90 dB    | 大声对话 / 摩托车噪音，多小时暴露可损伤 |               |
| 100 dB   | 电吹风、喇叭声，数分钟即可有风险      |               |
| 110 dB   | 警笛 / 大型机械，数分钟足以造成损伤   |               |
| ≥ 120 dB | 爆竹 / 飞机声，极短时间也会造成即时损伤 | ([世界卫生组织][1]) |

## 🧠 三、为什么声音会损伤听力？

噪音主要伤害**内耳的毛细胞**，这些细胞一旦受损不会再生。当声音强度高或暴露时间长时：

* 毛细胞受到机械性损伤
* 听觉神经刺激过度
* 导致噪音性听力损失（NIHL）
* 有时伴随耳鸣、听觉敏感性下降 ([gesund.bund.de][5])

---

## 📉 四、日常噪音对健康的其他负面影响

除听力损伤外，长期暴露于过高噪音还可能：

* 干扰睡眠、学习与集中力
* 提高心血管问题风险
* 增加心理压力和烦躁感
* 儿童语言理解与发展受影响 ([世界卫生组织][6])
