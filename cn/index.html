如何使用本分贝计

  1. 1
    允许麦克风访问

    查看实时 dB 数值以及颜色提示。建议持续测量约 1 分钟，以过滤偶发的噪音峰值。
  3. 3
    停止测量并保存报告
    点击“停止测量”（按钮会自动切换），然后点击“保存测量报告”即可生成 PNG 图片，方便留档或分享。
  4. 4
    根据结果采取行动
    当噪音水平偏高时，尽量缩短暴露时间、佩戴耳塞或耳罩，并远离声源以保护听力。

不同分贝对应的安全暴露时间

根据 NIOSH/CDC 的建议，噪音越大，可接受的安全暴露时间就越短。

70 dB 暴露时间不限

通常对全天暴露安全。

85 dB 最长约 8 小时

NIOSH 建议的 8 小时职业暴露上限。

90 dB 最长约 2–4 小时

类似割草机、摩托车等噪音水平。

100 dB 最长约 15 分钟

如吹风机、高功率电动工具等。

110 dB 最长约 1–2 分钟

摇滚演唱会、近距离警笛声等。

120+ dB 可能立即造成损伤

如鞭炮、喷气发动机等，应尽量避免暴露。

⚠️ 重要提示： 每增加 3 dB，安全暴露时间大约减半。例如 85 dB≈8 小时、88 dB≈4 小时、91 dB≈2 小时。 详情可参考 NIDCD 官方说明。

理解分贝与声音测量

分贝计（声级计）用于测量声音压力级（SPL），单位为 dB。本在线工具通过设备麦克风采集环境噪音，并给出实时测量结果。

为什么要关注分贝？ 分贝是对数刻度：每增加 10 dB，声强约增加 10 倍，主观听感大约“响一倍”。这意味着 80 dB 并不是只比 70 dB 稍微大一点，而是强度明显更高。

🏥 WHO 建议： 世界卫生组织 建议长时间环境噪音尽量低于 70 dB，以减少听力受损风险。持续暴露在超过 85 dB 的环境中，可能导致不可逆的听力损伤。

噪音如何损伤听力？ 根据多方健康机构的说明，强噪音会损伤内耳中细小而脆弱的毛细胞，这些细胞一旦受损无法再生，属于永久性损伤。高强度或长时间噪音可能导致：

  • 内耳毛细胞的机械性损伤
  • 听觉神经长期过度刺激
  • 噪音性听力损失（NIHL）
  • 耳鸣、对声音敏感等问题

在线分贝计的常见使用场景

本工具可以让您的浏览器在一定程度上充当“噪音检测仪”。虽然无法完全替代经过校准的专业级声级计，但在日常生活和工作场景中，它可以帮助您对环境噪音有一个直观的认识，从而更好地保护听力和健康。

🏠 家居与生活

  • 家电噪音测试： 检查新购电器（如洗碗机、吸尘器）是否如宣传那样安静。
  • 邻居噪音记录： 客观记录楼上装修、深夜音乐等噪音水平，为沟通提供依据。
  • 睡眠环境评估： 确保卧室夜间噪音维持在 30–40 dB 左右，有利于深度睡眠。

💼 办公与学习

  • 开放办公区域： 找出噪音高峰时段，以便安排需要高度专注的工作任务。
  • 自习环境选择： 在图书馆或咖啡馆中，寻找噪音在 40–50 dB 左右的角落。
  • 远程会议前测试： 在参加重要视频会议前，确认背景噪音处于可接受范围。
  • 合规记录： 使用“保存测量报告”功能，为工作场所噪音记录提供视觉证据。

🎧 音频与娱乐

  • 安全听音： 调整音响或耳机音量，使长期听音保持在 70–80 dB 以内。
  • 演出与活动： 在演唱会、酒吧等场合检测现场音量是否超过 100 dB。
  • 设备噪音： 测试电脑风扇或游戏主机在高负载时的噪音水平。

🩺 健康与舒适度管理

  • 耳鸣管理： 帮助识别容易引发耳鸣或不适的噪音环境。
  • 婴幼儿环境： 确保婴儿房安静舒适，白噪音机音量不过大。
  • 压力管理： 观察噪音水平与自己心情、专注度之间的关系，调整日常环境。

技术原理：基于浏览器的音频分析

与使用专用硬件传感器的传统声级计不同，本工具基于现代浏览器内置的 Web Audio API 标准工作。大致流程如下：

  1. 信号采集： 浏览器向系统请求麦克风输入权限。
  2. 数字采样： 将连续的声波转换为每秒数千次采样的数字信号。
  3. 频谱分析： 使用快速傅里叶变换（FFT）分析不同频率的能量分布。
  4. RMS 计算： 计算均方根（RMS）幅度，得到音频信号的平均功率。
  5. 映射到 dB： 通过校准算法，将原始数值转换为近似的分贝读数。

关于校准： 不同设备（手机、笔记本、外接麦克风）的灵敏度差异很大，因此网络版分贝计无法在未经校准的情况下做到绝对准确。 我们采用了针对常见消费级设备的“最佳拟合曲线”，以在相对测量上尽量接近真实环境。

常见问题

这个在线分贝计适合做什么？

它非常适合日常参考和趋势判断，例如比较不同房间或不同时段的噪音水平。但对于严格的职业安全检测或法律用途，仍建议使用经过校准的专业级声级计。

噪音是如何损伤听力的？

强噪音会损伤内耳耳蜗中微小的毛细胞。这些细胞负责将声波振动转化为大脑能接收的电信号。这些细胞一旦受损便无法再生，从而导致永久性听力损失。高强度或长时间的噪音暴露会加速这种损伤。如果您担心您的听力健康，请考虑参加我们免费的在线听力测试。

为什么需要允许麦克风访问？

因为只有通过麦克风才能实时采集环境噪音。本工具不会录音，也不会上传原始音频数据，所有处理仅在您的浏览器本地完成。

什么样的噪音水平算“安全”？

综合 NIOSH/CDC 等机构的建议：长期噪音尽量低于 70 dB 为宜；职业环境下 85 dB（8 小时）被视为上限。超过 85 dB 的环境，应缩短暴露时间并佩戴听力防护。

查看 NIOSH 指南 →

手机和电脑上测得的结果会不同吗？

会。不同设备的麦克风灵敏度、位置、指向性都会影响读数。如果您需要对比，请尽量使用同一台设备、相似的摆放位置进行测量。

“每增加 3 dB 暴露时间减半”是什么意思？

这就是所谓的 3 dB 交换率：在 85 dB 下推荐最长 8 小时暴露；如果噪音提升到 88 dB，则安全时间减半为约 4 小时；91 dB 为约 2 小时，以此类推。这说明看似不大的分贝变化，对听力风险影响很大。
