騒音レベルの目安

0〜40 dB - 静か
図書館 (30 dB)、ささやき声、木の葉の音など。どれだけ長時間でもほぼ安全です。
40〜70 dB - 適度
通常の会話 (60 dB)、静かなオフィスなど。一般的には長時間の暴露も問題ありません。
70〜85 dB - 注意
交通量の多い道路 (80 dB)、掃除機など。長時間続くと疲労やストレスの原因となります。
85〜100 dB - 有害
電動工具 (90 dB)、バイクのエンジン音など。8 時間を超える暴露は聴覚障害のリスクがあります。
100〜120 dB - 非常に有害
チェーンソー (110 dB)、サイレンなど。数分の暴露でもダメージの原因となります。
120+dB - 危険
ジェット機のエンジン (130 dB)、花火の至近距離など。瞬時に聴覚を損なう危険があります。

補足: デシベルは対数スケールです。10 dB 上がるごとに音のエネルギーは約 10 倍になり、体感的にもかなり大きくなります。

使い方

  1. 1
    マイクアクセスを許可する
    「測定を開始」をクリックし、ブラウザのダイアログでマイクへのアクセスを許可します。
  2. 2
    騒音レベルを確認する
    リアルタイムの dB 値と色分けされたバーで騒音レベルを確認できます。おすすめ: 少なくとも 1 分ほど測定して、瞬間的なピークをならしましょう。
  3. 3
    停止してレポートを作成
    「測定を停止」後に「レポートを保存」をクリックすると、そのセッションの統計をまとめた PDF をダウンロードできます。
  4. 4
    結果を活用する
    測定結果を活かして、耳に負担をかける音源を特定し、騒音対策や生活環境の改善に役立ててください。

デシベルと音の測定について

騒音計 (デシベルメーター) は、音の大きさを表す指標である音圧レベル (SPL) をデシベル (dB) で測定する機器です。このオンラインツールは、お使いのデバイスのマイクを利用して周囲の音を検出し、リアルタイムに騒音レベルを表示します。

デシベルが重要な理由: デシベルスケールは対数的なため、数値が少し変化しただけでも実際のエネルギー量は大きく変わります。10 dB 増えるごとに音のエネルギーは約 10 倍になり、体感的にも「かなり大きくなった」と感じるレベルです。

🏥 WHO の推奨値: 世界保健機関 (WHO) は、長時間にわたる騒音暴露を避けるため、日常生活では 70 dB 未満 を目安とすることを推奨しています。85 dB を超える環境では、暴露時間が長くなるほど聴覚障害のリスクが高まります。

騒音が聴覚を傷つける仕組み: 各国の公的機関 によると、強い音に長時間さらされると、内耳にある繊細な有毛細胞が損傷します。これらの細胞は再生しないため、一度失われた聴力は元に戻りません。具体的には次のような影響が生じます。

  • 内耳の有毛細胞の物理的な損傷
  • 聴神経の過剰な刺激
  • 騒音性難聴 (NIHL) の発症
  • 耳鳴りや音に対する過敏症状

プライバシーとデータ保護

プライバシーは最優先です。 音声データの処理はすべてブラウザ内で完結し、録音・保存・サーバーへの送信は行われません。「レポートを保存」機能も、現在のセッションの統計値 (最小・最大・平均) を使って、デバイス上で直接 PDF を生成します。

オンライン騒音計の実用的な活用例

この無料オンライン騒音計は、ブラウザを使って身近な環境の騒音レベルを簡単に把握できるツールです。校正されたクラス 1 の専門機器ほどの精度はありませんが、日常生活や仕事・学習環境で「どの程度うるさいか」を把握するには非常に有用です。

🏠 自宅・日常生活

  • 家電の騒音チェック: 新しく購入した洗濯機や掃除機が、カタログどおりの静音性かどうかを確認。
  • 近隣騒音の記録: 上下階や周囲の部屋からの生活音や工事音の大きさを客観的に測定。
  • 睡眠環境の最適化: 寝室の騒音レベルが快適な睡眠に適した 30〜40 dB 程度かどうかをチェック。

💼 仕事・学習環境

  • オフィスの騒音管理: オープンスペースの騒音レベルを測定し、集中作業に適した時間帯を把握。
  • 自習スペースの選定: カフェや図書館など、どの席が最も静かかを比較。
  • オンライン会議前のチェック: 会議前に背景騒音レベルを測定し、マイクテストとして活用。
  • 記録用レポート: 「レポートを保存」機能を使って、職場の騒音状況を PDF として保存・共有。

🎧 音楽・エンタメ・機器設定

  • 安全な音量設定: スピーカーやヘッドホンの音量を調整し、映画鑑賞時に 70〜80 dB 程度に収まるようにする。
  • イベント会場の確認: ライブハウスやクラブの騒音レベルが危険域 (100 dB 超) に達していないかを確認。
  • PC・ゲーム環境: ゲーミング PC や空調機器のファンノイズを測定し、静音化の指標とする。

🩺 健康・ウェルビーイング

  • 耳鳴り (耳鳴り) 管理: 耳鳴りが悪化しやすい環境の騒音レベルを把握し、回避の参考にする。
  • 乳幼児の環境確認: 赤ちゃんのいる部屋の騒音が過度になっていないか、ホワイトノイズマシンの音量が大きすぎないかをチェック。
  • ストレス要因の可視化: ストレスを感じやすい時間帯・場所と騒音レベルを関連づけて把握し、生活改善に役立てる。

ブラウザでの音声解析の仕組み

一般的な専用騒音計は、専用センサーと校正済み回路を用いて音圧レベルを測定しますが、このツールはモダンブラウザに搭載されている Web Audio API を利用して同様の計測をソフトウェア的に実現しています。処理の流れは次のとおりです。

  1. 信号の取得: ブラウザがマイクへのアクセス許可を要求し、音声入力ストリームを取得します。
  2. デジタル変換: 連続的な音の波は、毎秒数千回という頻度でサンプリングされ、デジタルデータに変換されます。
  3. 周波数解析: FFT (高速フーリエ変換) と呼ばれるアルゴリズムで、周波数帯ごとのエネルギー分布を解析します。
  4. RMS 計算: RMS (二乗平均平方根) を計算し、一定時間内の平均的な音の大きさを求めます。
  5. dB スケールへの変換: 得られた数値を、一般的な室内環境を想定したキャリブレーションカーブに基づき、デシベル値として画面に表示します。

キャリブレーションについて: スマートフォン・ノート PC・外付けマイクなど、デバイスによってマイクの感度は大きく異なります。そのため、外部校正のないウェブベースの騒音計が 100% の絶対精度を持つことはありません。本ツールでは、一般的なコンシューマーデバイスのマイクで使いやすいように調整された「おおよその目安」を提供しています。

よくある質問

この騒音計はどの程度正確ですか？

オンライン騒音計は実用的な目安を提供しますが、専門の測定器と完全に一致するとは限りません。精度はマイクの品質や環境条件、デバイスごとの設定に左右されます。法的な証拠や厳密な測定が必要な場合は、校正済みの測定器を使用してください。

デシベルレベルの目安を見る →

なぜマイクへのアクセス許可が必要ですか？

騒音レベルを測るためには周囲の音をマイクから取得する必要があります。音声データはブラウザ内だけで処理され、録音・保存・送信は行われません。プライバシー保護のため、必要最小限のアクセスのみを使用しています。

どのくらいの音量なら安全ですか？

NIOSH/CDC によると、70 dB 未満 の騒音は長時間の暴露でも一般的に安全とされています。職場における推奨限度は 85 dB・8 時間 で、それ以上のレベルに長期間さらされると聴覚障害のリスクが高まります。

NIOSH のガイドライン →

スマートフォンやタブレットでも使えますか？

はい。iOS Safari や Chrome など、最新のブラウザを備えたスマートフォンやタブレットで利用できます。ブラウザがマイクアクセスを求めたときに許可すると、測定が開始されます。端末によってマイク性能が異なる点にご注意ください。

騒音はどのようにして耳にダメージを与えますか？

強い音にさらされ続けると、内耳の有毛細胞が損傷します。これらの細胞は一度傷つくと再生しないため、聴力は元に戻りません。音が大きいほど、また暴露時間が長いほど、ダメージが蓄積しやすくなります。

NIDCD による詳細解説 →

「3 dB 交換率」とは何ですか？

NIOSH の 3 dB 交換率とは、騒音レベルが 3 dB 上がるごとに安全な暴露時間が半分になるという考え方です。たとえば、85 dB で 8 時間、88 dB で 4 時間、91 dB で 2 時間というように、わずかな数値の違いでも許容できる暴露時間が大きく変わります。
